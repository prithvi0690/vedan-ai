## Objective
Build a verified AI system that answers GST, tax, and compliance queries
using only official government documents.

## Functional Requirements
- Accept natural-language queries from users.
- Classify queries by domain (GST, Income Tax, Schemes).
- Retrieve relevant information only from official documents.
- Generate answers with clear source citations.
- Allow users to view referenced government documents.

## Non-Functional Requirements
- High accuracy with zero hallucination.
- Low response time for queries.
- Scalable document ingestion and retrieval.
- Secure handling of user inputs.

## Data Requirements
- Government PDFs (GST Acts, notifications, circulars).
- Metadata including document source and effective date.

## Current Stage
Proof of Concept (POC).
